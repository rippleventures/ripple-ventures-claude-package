# Calibration log

Each file captures a run where the skill missed something or the user had to correct the output.

## How to use

When the user flags a miss, copy `_template.md`, fill in, save as `YYYY-MM-DD-short-slug.md`. The skill reads recent entries on every run.

## What gets logged

- Output that consistently missed the mark
- Inputs the skill failed to ask for upfront
- Format / structure preferences specific to the user's firm

Don't log:

- One-off mistakes with no transferable lesson
- Stylistic preferences (those go in `references/`)
