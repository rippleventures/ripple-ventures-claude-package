# Calibration log

Each file captures a case where the skill miscalled something — labeled a term as market when it wasn't, missed a red flag, or asked the wrong clarifying question.

## How to use

After a review or drafting task, if the user flags a miss, copy `_template.md`, fill in the sections, save as `YYYY-MM-DD-short-slug.md`.

Slug examples:

- `2026-05-08-3x-liq-pref-context`
- `2026-05-12-mfn-cap-missed`
- `2026-05-15-ccpc-flag-acumen`

## What gets logged

- Patterns where framework standards diverge from the user's actual closed-deal precedent
- Red flags the skill consistently misses
- Cross-border issues that need extra scrutiny in specific deal types
- Counter-positions that worked vs. didn't — which negotiations succeeded with which framing

Don't log:

- Random one-off mistakes with no transferable lesson
- Stylistic preferences (those go in the SKILL.md tone section)
