# Calibration log

Each file captures a run where the skill produced output that didn't match the user's firm's conventions.

## How to use

When the user flags formatting or structure issues, copy `_template.md`, fill in, save as `YYYY-MM-DD-short-slug.md`.

## What gets logged

- Firm-specific formatting conventions (color coding variations, sheet naming, etc.)
- Default sheet structures the user prefers
- QC checks the skill should add to the standard pass

Don't log:

- One-off mistakes with no transferable lesson
