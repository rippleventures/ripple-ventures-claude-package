# Calibration log

Each file captures a brief that missed something or miscategorized someone. The skill reads these on every run, newest first.

## How to use

When the user flags a miss, copy `_template.md`, fill in, save as `YYYY-MM-DD-short-slug.md`.

Slug examples:

- `2026-05-08-acme-call`
- `2026-05-12-luma-event-mis-bucketed`
- `2026-05-15-cold-outreach-batch`

## What gets logged

- Sources the skill should have checked but didn't
- Categories of past interaction the skill consistently misses
- Person Type misclassifications in list mode
- Conversation topics that landed vs. fell flat
- LinkedIn-blocked patterns the skill kept tripping over

Don't log:

- One-off factual errors with no transferable lesson
- Cases where insufficient data led to "Unknown" — that's expected
