# References

If you want a custom onboarding script (different questions, different ordering, role-specific flow), save it here as `custom-flow.md` and the skill will use it instead of the default flow in `SKILL.md`.

Useful when:

- Onboarding a specific role (e.g., a CFO vs. a product manager — different setup questions)
- Different industry templates (VC fund vs. operating company vs. consulting firm)
- A simplified flow for users who only want a subset of skills
